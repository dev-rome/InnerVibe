# InnerVibe

InnerVibe is a simple mood and reflection journal built with Next.js.  
The goal of the project is to help users track their mood and write short reflections over time.

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- NextAuth (Google OAuth)
- PostgreSQL
- Prisma (ORM)
- Tailwind CSS
- Framer Motion
